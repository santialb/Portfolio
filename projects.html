Fixed a critical bug in the "Captain's Log" exercise test suite that was causing incorrect test validations for ship registry numbers.

🐛 Issue Fixed:

The randomShipRegistryNumber() method was generating numbers outside the valid range (1000-9999), causing tests to fail unexpectedly.

✅ Solution Implemented:

String randomShipRegistryNumber() {
                                    var start = 1000;
                                -   var end = 11000;
                                +   var end = 10000;
                                    return String.format("NCC-%d", start + random.nextInt(end - start));
                                }

  • Fixed test validation for thousands of students learning Java
  • Ensured ship registry numbers stay within the correct range (NCC-1000 to NCC-9999)
